# 个人主页网站

一个简洁、现代化的个人主页，采用纯HTML + CSS + JavaScript构建。

## 在线预览

https://dapper-kitten-222be8.netlify.app/

## 主要功能

- 📱 完全响应式设计
- 🎨 深色/浅色主题切换
- 🖼️ WebP图片优化
- ⚡ 图片懒加载
- ✨ 页面加载和滚动动画
- 📊 Google Analytics支持

## 技术栈

- HTML5
- CSS3（Flexbox、Grid、CSS变量）
- JavaScript（原生，无依赖）
- Font Awesome图标库

## 本地运行

直接用浏览器打开 `index.html` 文件即可。

或使用本地服务器：
```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```

## 部署

项目已部署在Netlify，可以直接推送到GitHub后自动部署。

## 项目结构

```
my-website/
├── index.html              # 主页面
├── style.css               # 样式文件
├── images/                 # 图片资源
├── README.md               # 项目说明
├── CHANGELOG.md            # 更新日志
└── convert_to_webp.py      # 图片转换工具
```
